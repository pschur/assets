# Laravel Webfiles

## Nutzung
kopiere dir den Html-Code:
```html
<link rel="stylesheet" href="https://raw.githubusercontent.com/pschur/assets/main/laravel/css/app-<ID>.css">
<script src="https://raw.githubusercontent.com/pschur/assets/main/laravel/js/app-<ID>.js" defer></script>
```
Ersetze nun ``` <ID> ``` durch eine ID aus der Liste


| ID   | Beschreibung  |
| --------- | ------- |
| 01 | hier ist Tailwindcss mit [turbolinks](https://github.com/turbolinks/turbolonks) |
